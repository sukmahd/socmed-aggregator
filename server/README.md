# twatt-api


Demo app with basic REST API and Twitter API OAUTH

## REST API TWITTER

List of customer routes:

| Route             | HTTP          | Description      |
| -------------     |:-------------:| :----------------|
| /twatt/search       |GET            | Get all search info    |
| /twatt/timeline     |GET            | Get timeline info     |
| /twatt/post         |POST           | Post a tweet|
